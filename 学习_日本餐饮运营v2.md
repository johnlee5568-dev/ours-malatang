### 之前的错误思维（中国餐饮）
- ❌ 双人套餐
- ❌ 多人优惠

## 二、日本餐饮基础认知

### 1. 日本人消费习惯

| 习惯 | 说明 |
|------|------|
| **割り勘** | 各付各的，极少请客 |
| **一行筷子** | 一群人去，各点各的 |
| **午餐一人** | 一个人去拉面店、定食店很正常 |
| **网络依赖** | 找店靠Tabelog，预约靠Hot Pepper |
| **评价敏感** | Tabelog 3.5分以下基本不去 |

### 2. 日本餐饮分类（日文）

| 中文 | 日文 | 消费场景 |
|------|------|----------|
| 定食 | 定食 | 午餐一人食 |
| 拉面 | ラーメン | 午餐、夜宵 |
| 居酒屋 | 居酒屋 | 晚餐、饮酒 |
| 火锅 | 鍋 | 冬季、聚餐 |
| 中华 | 中華 | 晚餐、纪饿 |

### 3. 日本人找店流程

```
1. Google搜索 / Tabelog评分
   ↓
2. 查看菜单、图片、评价
   ↓
3. Hot Pepper预约（多数店）
   ↓
4. 前往用餐
```

## 三、日本餐饮营销方式

### 1. 平台运营

| 平台 | 作用 |
|------|------|
| **Tabelog** | 评分、评价、发现 |
| **Hot Pepper** | 预约、优惠券、积分 |
| **Google Maps** | 位置信息、评价 |
| **Instagram** | 视觉营销、打卡 |

### 2. 常见促销方式

| 方式 | 日文 | 说明 |
|------|------|------|
| 午市套餐 | ランチセット | 低价套餐吸引午餐 |
| 单品优惠 | 単品割引 | 特定菜品打折 |
| 积分会员 | ポイント | 消费积累 |
| 集章卡 | スタンプラリー | 吃5次送1次 |
| 首次优惠 | 初めて割 | 新客discount |

### 3. 不适合日本的方式

- ❌ 双人套餐（日本人各付各）
- ❌ 三人同行一人免费（几乎无感）
- ❌ 放题/自助餐（成本风险高）
- ❌ 整桌打折（不符合AA习惯）

## 五、对OURS麻辣烫的建议

### 1. 菜品策略

| 方向 | 说明 |
|------|------|
| **一人食** | 午餐为主的定位 |
| **小份化** | 降低单价，让一人也能轻松点 |
| **辣度选择** | 明确分级：不辣→微辣→中辣→辣 |
| **好看优先** | 摆盘要精致，适合拍照 |

### 2. 活动策略

| 活动 | 形式 |
|------|------|
| **Instagram联动** | 发IG送小菜 |
| **集章卡** | 吃5次送1次 |
| **首次优惠** | 新客汤底半价 |
| **限定套餐** | 季节限定口味 |

### 3. 平台策略

| 平台 | 动作 |
|------|------|
| Tabelog | 开通并优化页面，争取4.0+评分 |
| Hot Pepper | 上线预约和优惠券 |
| Instagram | 每日更新，发高质量内容 |
| Google Maps | 确保信息准确，鼓励评价 |

## 六、总结

### 日本餐饮运营关键点

1. **个体思维** - 不是"一桌人"，是"一个个客人"
2. **场景思维** - 午餐/晚餐场景明确分开
3. **平台思维** - Tabelog + Hot Pepper + Instagram
4. **颜值思维** - 拍照好看 > 好吃
5. **评价思维** - 评分是生命线
