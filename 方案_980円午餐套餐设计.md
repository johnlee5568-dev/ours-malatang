## 三、为什么980円是合理的价格

### 消费者视角

| 价格 | 消费者感受 |
|------|------------|
| 700円以下 | "可能不太好吃" |
| **980円** | **"正好！一个人吃这个不错"** |
| 1200円+ | "有点贵 lunch有点浪费" |

### 竞争对手对比

| 店铺 | 午餐价格 |
|------|----------|
| 丸龟制面 | 780円 |
| 拉面店 | 850～950円 |
| **OURS 980円套餐** | **含饮料+小菜=超值的** |
| 吉野家/松屋 | 650～850円 |

**OURS的980円套餐 = 拉面价格 + 饮料 + 小菜 = 性价比高**

## 七、总结

### 980円套餐的意义

1. **市场定位准确** - 拉面同价，但内容更丰富
2. **价值感强** - 消费者觉得"买到就是赚到"
3. **翻台率高** - 制作简单，3分钟出餐
4. **吸引午餐客流** - 填补工作日空白
