# Ours麻辣湯 Hot Pepper运营优化方案

| 问题 | 日本消费者视角 |
|------|----------------|
| 评论只有2条 | "不知道好吃与否，不敢去" |

## 二、核心策略（日本思维）

### 1. 增加评论（最重要）

**日本餐饮的铁则：没有4.0+评分 = 无人来**

| 动作 | 方法 |
|------|------|
| 好评诱导 | 结账时说"如果方便的话，请写评论" |
| 回报 | 写评论送小菜或饮料 |
| 及时回复 | 每条评论必回复，展现诚意 |
| 邀请再访 | "期待下次光临" |

**具体话术**：
```
「お皿の片付けありがとうございます！
もし宜しければ、食べログかホットペッパーで
レビューを書いて頂けると幸いです。
 次回来店時にクーポン为您准备着♪」
```

### 2. 强调「一人ok」（お一人様）

**日本午餐的核心需求**

| 现状 | 优化方向 |
|------|----------|
| 现在：无强调 | → 加「一人ok」「お一人様歓迎」 |
| 现在：强调宴会 | → 午餐宣传改为一人都市 |

**页面关键词修改建议**：
- ランチ：お一人様OK
- 麻辣湯：1人前の量为ベース
- サイドメニュー：豊富な小皿料理

### 3. 平日午餐优惠

**日本人的消费习惯：平日はランチがお得**

| 活动 | 内容 |
|------|------|
| 平日ランチセット | 麻辣湯（小）+米饭+饮料 = 980円 |
| ホットペッパーCoupon | 平日限定：料理10%off |

### 4. 照片戦略

**日本人在Hot Pepper上最看重照片**

| 照片类型 | 建议 |
|----------|------|
| 料理写真 | 必须清晰、看起来好吃 |
| 店内写真 | 明るく清洁感 |
| 菜单 | 明确价格 |

## 五、总结：纯日本思维

### 不该做的

- ❌ 双人套餐（日本人各付各）
- ❌ 三人同行优惠（无感）
- ❌ 强调麻辣（日本人不太能吃辣）

### 应该做的

- ✅ 一人食套餐（午餐核心）
- ✅ 明确价格（ランチ980円这样）
- ✅ 评论诱导（4.0+评分）
- ✅ 周末预约引导（通过Hot Pepper）
- ✅ 照片好看（Instagram联动）
