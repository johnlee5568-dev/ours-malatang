# 日本餐饮运营学习笔记

## 一、日本餐饮市场特点

### 1. 消费者习惯

| 特点 | 说明 |
|------|------|
| 个付个 | 日本人聚餐习惯各自付款（割り勘） |
| 一人食 | 单身人口多，一人用餐需求大（13万件以上） |
| 网络预约 | 80%以上通过Hot Pepper/Tabelog预约 |
| 评价导向 | 非常看重评分（Tabelog 3.5以上才算好） |
| 场景消费 | 明确区分午餐/晚餐场景 |

### 2. 热门场景（从Hot Pepper观察）

| 场景 | 日文 | 需求 |
|------|------|------|
| 女子会 | 女子会 | 闺蜜聚餐 |
| 约会 | デート | 2人约会 |
| 一人食 | お一人様 | 单独用餐 |
| 生日/纪念日 | 誕生日・記念日 | 庆祝场合 |
| 欢送会/欢迎会 | 歓送迎会 | 宴会 |

### 3. 常用搜索条件

- ネット予約可（可网络预约）
- ランチ（午餐）
- ディナー（晚餐）
- 食べ放题（自助餐）
- 飲み放题（畅饮）
- 個室（包间）
- 禁煙/吸烟

## 四、对OURS麻辣烫的启示

### 1. 应该做的

| 动作 | 理由 |
|------|------|
| 上Tabelog/Hot Pepper | 日本人找店主要渠道 |
| 网络预约 | 符合日本消费习惯 |
| 一人食套餐 | 满足午餐需求 |
| 女士/闺蜜活动 | 符合日本消费场景 |
| 积分会员系统 | 符合日本消费习惯 |

### 2. 不适合做的

| 动作 | 理由 |
|------|------|
| 双人套餐 | 日本人习惯各自付款 |
| 放题模式 | 成本不可控 |
| 整桌优惠 | 不符合AA制习惯 |

### 3. 定价参考

日本中华料理店平均消费：
- 午餐：800-1500円
- 晚餐：1500-3000円
- 整体属于中低档消费
