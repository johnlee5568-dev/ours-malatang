## 三、活动方案

### 1. 女士之夜（每周五）

| 项目 | 内容 |
|------|------|
| 时间 | 每周五晚市 |
| 对象 | 女性顾客 |
| 活动 | 到店女士送甜品或小菜（价值200円） |
| 目的 | 吸引年轻女性，增加夜间客流 |

### 2. 闺蜜优惠（每周日）

| 项目 | 内容 |
|------|------|
| 时间 | 每周日全天 |
| 对象 | 3人及以上顾客 |
| 活动 | 3人同行1人免费（相当于75折） |
| 目的 | 吸引闺蜜聚餐，提高客单价 |

| 对象 | 首次到店顾客 |
